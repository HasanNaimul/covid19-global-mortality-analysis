# LinkedIn Project Entry

**Title:** Global COVID-19 Mortality Analysis

**Description:**
Built an end-to-end data analysis project exploring COVID-19 mortality trends across six countries using daily time-series data. Cleaned and transformed raw data, engineered rolling metrics and country-level KPIs, detected mortality waves, measured post-peak recovery speed, segmented countries with clustering, and created both static visualizations and an interactive dashboard to communicate insights.
